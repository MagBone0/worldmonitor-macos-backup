## 📋 About worldmonitor-macos-backup

worldmonitor-macos-backup is a native macOS app that shows current geopolitical data. It tracks cyber threats, space weather events, disease outbreaks, air quality, and AI news on over 50 live data panels. The app uses modern technology including Tauri 2.x, TypeScript, Rust, and Vite to provide accurate real-time information.

This dashboard gives you a clear picture of global risks and events as they happen. It is designed to run on macOS systems, using local resources efficiently while pulling live data through secure connections.

## 🛠 Troubleshooting

If you have issues, try these steps:

- Make sure your macOS version is 11 or newer.
- Confirm you have a stable internet connection.
- Restart the app.
- Check permissions in **System Preferences > Security & Privacy > Privacy** to ensure the app can connect to the internet.
- Re-download the app if it crashes on startup or behaves abnormally.
- If panels won’t update, quit and reopen the app.
